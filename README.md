## 📖 Overview

SkillBridge AI is a modern AI-powered career platform that helps students prepare for placements, build ATS-friendly resumes, practice mock interviews, track coding progress, enroll in learning programs, apply for internships, and connect with recruiters through a unified dashboard.

Designed with a scalable architecture using **Next.js**, **TypeScript**, **Tailwind CSS**, and **Supabase**, the platform delivers a responsive, production-ready experience with role-based access for Students, Recruiters, Placement Officers, and Administrators.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/balareddy12/Skill_Bridge.git
```

## Navigate

```bash
cd Skill_Bridge
```

## Install Dependencies

```bash
npm install
```

## Configure Environment

Create a `.env.local` file:

```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```

## Start Development Server

```bash
npm run dev
```

Visit:

```
http://localhost:3000
```
